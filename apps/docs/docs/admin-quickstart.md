title: Admin Quickstart
---

Steps:

1. Create a Discord server and a bot application
2. Create roles: admin, member, contributor
3. Invite the bot with Bot and applications.commands scopes
4. Map role IDs to env vars
5. Set core API URL and key
6. Run command registration

Commands:

- /verify wallet: verifies a wallet and updates roles
- /status: shows current membership status and roles
- /refresh-roles: reconciles roles to core state
