Steps: <!-- IC: 295 -->

1. Create a Discord server and a bot application <!-- IC: 296 -->
2. Create roles: admin, member, contributor <!-- IC: 297 -->
3. Invite the bot with Bot and applications.commands scopes <!-- IC: 298 -->
4. Map role IDs to env vars <!-- IC: 299 -->
5. Set core API URL and key <!-- IC: 300 -->
6. Run command registration <!-- IC: 301 -->

Commands: <!-- IC: 302 -->

- /verify wallet: verifies a wallet and updates roles <!-- IC: 303 -->
- /status: shows current membership status and roles <!-- IC: 304 -->
- /refresh-roles: reconciles roles to core state <!-- IC: 305 -->
