title: Extension Points and Roadmap
---

Planned areas:

- Advanced moderation, appeals, and case management
- Rich notification workflows and templating
- Cross-platform chat integrations beyond Discord
- Event platform and on-chain event hooks
- DAO governance integrations and proposal-based gating
- Webhook ecosystem with robust verification and retries
- Orchestration for multi-guild, multi-community deployments

Stubs in this repo:

- webhook-utils export a simple verifySignature to be replaced later
- integration-client can expand to cover more core endpoints
- Role mapping can evolve for dynamic rules and hierarchies
